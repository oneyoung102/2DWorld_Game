# 2DWorld_Game

> 본 파일은 `<windows.h>` 라이브러리를 사용하여 window 운영체제에서만 작동합니다.

> 현재 게임명은 가제로 처음 이름이 지어졌을 때부터 변경되지 않았습니다.   
> 파일 분리가 완전히 이루어지지 않았습니다.   
> 2D 월드에서 자신만의 캐릭터를 키우고 몬스터를 처치하는 게임입니다.

<hr/>

## 플레이

> **2DWorld.cpp**가 메인파일로 다른 파일과 함께 같은 프로젝트에 넣고, 실행시켜 터미널에서 플레이하세요.

  - #### 레벨 시스템   
  - #### 원소
  - #### 몬스터
      > 모든 몬스터는 특성과 약점, 강점을 갖습니다.
  - #### 플레이어   
      - **조작**   
        - 이동   
            `a`, `d`를 눌러 왼쪽, 오른쪽으로 움직입니다.   
            `A`, `D`를 눌러 왼쪽, 오른쪽으로 회피합니다.   
            `space bar`를 눌러 점프합니다.      
        - 회복      
            `w`를 눌러 회복합니다.   
            체력 포션이 있어야 합니다.   
            체력 포션을 하나 소모합니다.      
        - 상호작용   
            `q`를 눌러 상호작용합니다.
            - 공격   
              공격 시에 `x`를 통해 공격 범위가 표시됩니다.   
              공격 범위 안에 있는 몬스터는 공격받습니다.      
            - 체력 포션 획득   
              체력 포션 상자는 `☏`로 표시됩니다.   
              근처에서 상호작용하여 체력 포션을 획득하세요.
            - 멥 이동   
              멥의 양쪽 끝에서 상호작용하여 다음 멥으로 이동합니다.   
      - **인벤토리**   
        - 아이템   
        - 장비   
        - 이동하기   
  - #### 월드
